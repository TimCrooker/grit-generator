## Usage

Install [Projenerator](https://github.com/TimCrooker/projenerator) first.

```bash
yarn global add projenerator
# or
npm i -g projenerator
```

### From npm

```bash
projen <%= name.replace('projen-', '') %> my-project
```

### From git

```bash
projen <%= username %>/<%= name %> my-project
```
